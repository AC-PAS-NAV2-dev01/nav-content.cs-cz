## <a name="to-create-and-connect-an-incoming-document-record-from-a-purchase-invoice"></a><span data-ttu-id="b9a65-110">Vytvoření a připojení záznamu došlého dokladu k nákupní faktuře</span><span class="sxs-lookup"><span data-stu-id="b9a65-110">To create and connect an incoming document record from a purchase invoice</span></span>
1. <span data-ttu-id="b9a65-111">V pravém horním rohu zvolte ikonu **Vyhledat stránku nebo sestavu**, zadejte **Nákupní faktury** a pak vyberte související odkaz.</span><span class="sxs-lookup"><span data-stu-id="b9a65-111">In the top right corner, choose the **Search for Page or Report** icon, enter **Purchase Invoices**, and then choose the related link.</span></span>
2. <span data-ttu-id="b9a65-112">Vyberte řádek pro nákupní fakturu, kterou chcete připojit k souboru a pak zvolte akci **Vytvořit došlý doklad ze souboru**.</span><span class="sxs-lookup"><span data-stu-id="b9a65-112">Select the line for a purchase invoice that you want to attach a file to, and then choose the **Create Incoming Document from File** action.</span></span>
3. <span data-ttu-id="b9a65-113">Alternativně, vyberte řádek pro nákupní fakturu, kterou chcete připojit k souboru a pak zvolte akci **Připojit soubor**.</span><span class="sxs-lookup"><span data-stu-id="b9a65-113">Alternatively, select the line for a purchase invoice that you want to attach a file to, and then choose the **Attach File** action.</span></span>
4. <span data-ttu-id="b9a65-114">V okně **Vložit soubor** vyberte soubor, který představuje daný došlý doklad a pak zvolte tlačítko **Otevřít**.</span><span class="sxs-lookup"><span data-stu-id="b9a65-114">In the **Insert File** window, select the file that represents the incoming document in question, and then choose the **Open** button.</span></span>

## <a name="to-remove-a-connection-from-an-incoming-document-record-to-a-posted-document"></a><span data-ttu-id="b9a65-120">Odebrání připojení ze záznamu došlého dokladu k zaúčtovanému dokladu</span><span class="sxs-lookup"><span data-stu-id="b9a65-120">To remove a connection from an incoming document record to a posted document</span></span>
<span data-ttu-id="b9a65-121">Můžete odebrat soubor připojený k nezaúčtovaným dokladům smazáním souvisejícího záznamu došlého dokladu.</span><span class="sxs-lookup"><span data-stu-id="b9a65-121">You can remove file attachments from non-posted documents at any time by deleting the related incoming document record.</span></span> <span data-ttu-id="b9a65-122">Pokud je doklad zaúčtovaný, pak musíte nejdřív odebrat připojení ze záznamu došlého dokladu.</span><span class="sxs-lookup"><span data-stu-id="b9a65-122">If the document is posted, then you must first remove the connection from the incoming document record.</span></span>

1. <span data-ttu-id="b9a65-123">V pravém horním rohu zvolte ikonu **Vyhledat stránku nebo sestavu**, zadejte **Došlé doklady** a pak vyberte související odkaz.</span><span class="sxs-lookup"><span data-stu-id="b9a65-123">In the top right corner, choose the **Search for Page or Report** icon, enter **Incoming Documents**, and then choose the related link.</span></span>
